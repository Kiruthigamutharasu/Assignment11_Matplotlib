# Assignment 11: Matplotlib – Core Plot Types & Visualization


## Dataset Used

**Dataset Name:** Sample Superstore Dataset

**Source:** Kaggle

**Link:**  https://www.kaggle.com/datasets/bravehart101/sample-supermarket-dataset?select=SampleSuperstore.csv

### Columns Utilized

* Ship Mode
* Segment
* Country
* City
* State
* Postal Code
* Region
* Category
* Sub-Category
* Sales
* Quantity
* Discount
* Profit

The dataset was read using the Python `csv` module to comply with the restriction of not using Pandas plotting functions.



## Implemented Tasks

The notebook contains the following visualization tasks:

1. **Line Plot – Sales Trend**

   * Display sales variation
   * Added title and axis labels

2. **Scatter Plot**

   * Relationship between Sales and Profit
   * Proper labels and title

3. **Bar Plot**

   * Vertical bar chart
   * Horizontal bar chart
   * Titles and labels for both

4. **Multiple Bar Chart**

   * Comparison of sales across different years
   * Proper bar alignment and legend

5. **Stacked Bar Chart**

   * Online vs Offline sales representation
   * Labels and legend included

6. **Histogram**

   * Distribution of Quantity column
   * Appropriate number of bins

7. **Pie Chart**

   * Market share based on Sub-Category
   * Percentage values displayed
